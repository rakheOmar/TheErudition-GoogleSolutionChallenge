# System rules for this repository

## Ground Rules

1. Before using unfamiliar libraries, use the Context7 MCP server to retrieve the latest documentation.
2. If writing Supabase queries, use the Supabase MCP server to verify the latest database schema and status.
3. Before writing code:
   - Decompose the problem
   - Create a step-by-step plan
   - Then implement the solution
4. Use Exa web search when external or recent information is required.
5. Avoid unnecessary comments. Only comment complex business logic.
6. Do not introduce new dependencies unless necessary.
7. Follow the existing project architecture. Do not create new top-level directories.
8. Before creating new functions, search the repository for existing implementations.
